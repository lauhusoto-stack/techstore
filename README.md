# TechStore Chile

## Integrante

Laura Soto

## Objetivo del proyecto

Desarrollar una tienda virtual de tecnología utilizando React, HTML, CSS y JavaScript. La aplicación permite visualizar productos, buscar productos y agregar productos al carrito de compras.

## Tecnologías utilizadas

* React
* Vite
* HTML5
* CSS3
* JavaScript ES6
* Visual Studio Code
* Git
* GitHub

## Estructura del proyecto

* Header: Logo, buscador, botones y carrito.
* Navbar: Menú de navegación.
* Banner: Imagen principal y botón de ofertas.
* ProductList: Lista de productos.
* ProductCard: Tarjeta de cada producto.
* Sidebar: Beneficios de la tienda.
* InfoBar: Información de cuotas y despachos.
* Footer: Información de contacto.

## Instalación

1. Descargar o clonar el proyecto.
2. Abrir la carpeta en Visual Studio Code.
3. Abrir una terminal.
4. Ejecutar el comando:

npm install

5. Iniciar el proyecto con:

npm run dev

6. Abrir el enlace que aparece en la terminal.

## Dificultades encontradas

* Organización inicial de los componentes.
* Conexión de las props entre componentes.
* Implementación del buscador y del carrito de compras.

## Conclusiones

Este proyecto permitió aprender a crear una aplicación web con React, utilizar componentes, manejar estados con useState, trabajar con archivos JSON y desarrollar una interfaz moderna y organizada.
